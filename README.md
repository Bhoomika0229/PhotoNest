# PhotoBooth (Django)

A simple PhotoBooth web application built using Django.

## Features
- User login and authentication
- Capture photos using device camera
- View previously captured photos
- Delete photos from gallery

## Technologies Used
- Django (Python)
- HTML, CSS, JavaScript
- SQLite

## How to Run
1. Clone the repository
2. Install Django
3. Run migrations
4. Start the server

```bash
python manage.py runserver
